reverse(): It is used to generate dynamic URL. reverse is used to get the URL path from the name of the URL pattern. Instead of hardcoding URLs (like "/books/10/"), you use the URL "name" ("book-detail") and "args" 
	 then reverse will generate the correct URL.

		tar te solve karnyasathi verbose_name_plural use kartat tya model chya class Meta madhe khalil dilya pramane.

class ProductGallery(models.Model):
    product = models.ForeignKey(Product, on_delete=models.CASCADE, default=None)
    image = models.ImageField(upload_to='store/products', max_length=255)
    class Meta:
        verbose_name = 'productgallery'
        verbose_name_plural = 'product gallery'
